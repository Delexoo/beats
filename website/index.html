Small by design

A focused set of controls for listening while you work.

  1. Floating widget Stays on top of any app, always within reach.
  2. Folder playlists Your music, organized on disk the way you like it.
  3. Download from web Save tracks from the web into your playlists.
  4. Hide & show Alt+\ or the top chevron keeps the interface out of the way.

Simple setup

Install, add music, and place the widget where it belongs.

  1. 1. Download Install Beats on Windows in one click.
  2. 2. Add your music Point to a folder or paste a link to download.
  3. 3. Place it anywhere Drag the album art to position the widget.
  4. 4. Hide when needed Alt+\ or the top chevron keeps your screen clear.
