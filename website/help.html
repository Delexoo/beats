Help & manual

Everything you need to install Beats, build playlists, download music, and control the floating widget on Windows.

Getting started

Install once, create a playlist, add music, and press play.

  1. 1
    Install Beats Download and run Beats-Setup-x64.exe from the home page or GitHub Releases. Works on Windows 10 and 11 (64-bit).
  2. 2
    Open the dashboard Click the settings gear on the widget pill, or press Ctrl + \.
  3. 3
    Create a playlist Click New Playlist in the sidebar and give it a name. Each playlist is a folder on your disk.
  4. 4
    Add music Click the + on your playlist row, then choose Add files, Add folder, or Paste audio link.
  5. 5
    Press play Double-click a song in the list, or use the play/pause button on the widget.
First download? Beats automatically downloads yt-dlp, ffmpeg, and (for Spotify) spotDL on first use — about 130 MB total, one time only.

The widget

A small floating pill that stays on top of every app.

The widget shows album art, the current track, and playback controls. It has no taskbar icon and stays visible over other windows so you can control music without switching apps.

Move the widget

Drag the album art to reposition the pill anywhere on screen.

Hide and show

Press Alt + \ to slide the widget off the nearest screen edge. Press the same shortcut again to bring it back. You can also click the small chevron tab at the top center of your primary monitor.

Expanded controls

Hover the widget to reveal loop and settings buttons. Enable Extended widget in the dashboard header to keep those controls visible at all times.

Minimize

Click the minimize button in the dashboard to hide the widget completely. Use the top chevron or Alt + \ to show it again.

Dashboard

Your control center for playlists, tracks, and settings.

Open the dashboard with Ctrl + \ or by clicking the settings gear on the widget. It includes:

  • Playlists sidebar — create, rename, delete, and pin playlists
  • Track list — browse, play, reorder, and remove songs
  • Now playing footer — seek bar, volume, like/save, shuffle, and loop
  • Header controls — extended widget toggle, layout reset, help, donate, minimize, and exit

Resize the dashboard

Drag the edges to resize. Press Ctrl + Shift + \ to reset the widget position and dashboard size to defaults.

Zoom the track list

Hold Ctrl and scroll to scale song rows up or down.

Liked Songs & Saves

Use the heart and bookmark icons in the footer to add tracks to Liked Songs or Saves. These collections appear at the bottom of the playlist sidebar.

Playlists

Simple folder-based libraries you own completely.

Every playlist is a folder. By default they live in My Music\Beats. You can change the root folder in settings.

Create & manage

  • New Playlist — creates a new folder
  • Rename — right-click a playlist
  • Delete — right-click a playlist (removes the folder and its files)
  • Pin — right-click to pin a playlist above Liked/Saves

Add tracks

Click the + on a playlist row:

  • Add files — pick individual songs from your computer
  • Add folder — import an entire folder at once
  • Paste audio link — download from YouTube, Spotify, TikTok, SoundCloud, and more

Supported audio formats

MP3, WAV, FLAC, M4A, AAC, OGG, Opus, WMA, AIFF, and ALAC.

Tip: You can add or remove files directly in File Explorer. Beats watches playlist folders and updates the track list automatically.

Downloads

Paste a link and save audio straight into a playlist.

From any playlist, click + → Paste audio link, paste a URL, and click Download. Files are saved as MP3 in that playlist's folder.

Supported platforms

  • YouTube — videos, playlists, and channels
  • Spotify — tracks, albums, and playlists (via spotDL)
  • TikTok, SoundCloud, Instagram, and 1,000+ other sites via yt-dlp

Playlists & privacy

For YouTube and Spotify playlists, make sure they are public or unlisted. Private playlists cannot be downloaded.

Batch downloads

When downloading a playlist or album, Beats shows progress for each track. Large batches may take several minutes depending on your connection.

Shortcuts

Control Beats without touching the mouse.

Shortcut Action
Alt + \ Hide or show the widget
Ctrl + \ Open or close the dashboard
Ctrl + Shift + \ Reset widget position and dashboard size
Ctrl + scroll Zoom the track list (in dashboard)
Drag album art Move the widget
Top chevron tab Same as Alt + \
Hotkeys use the backslash (\) key on US keyboard layouts. If another app already registered a combo, use the on-screen buttons or chevron tab instead.

YouTube cookies

Optional setup when YouTube blocks downloads or asks you to sign in.

Most downloads work without any extra setup. If YouTube returns a sign-in or bot-check error, export a Netscape-format cookies.txt file while logged into YouTube and point Beats at it.

Step 1 — Export cookies

  1. Install a browser extension that exports Netscape-format cookies (see the yt-dlp wiki for recommended extensions).
  2. While logged into YouTube in your browser, export cookies for youtube.com.
  3. Save the file as cookies.txt somewhere permanent (e.g. your Documents folder).

Step 2 — Point Beats at the file

  1. Open the dashboard and click + on a playlist → Paste audio link.
  2. Click Set up cookies in Beats at the bottom of the dialog.
  3. Click Browse… and select your cookies.txt file.

Beats passes this file to yt-dlp on every download, so your browser can stay closed and you avoid "cookie database locked" errors from Edge or Chrome.

When to use this: Only when downloads fail with sign-in, bot-check, or age-restriction errors. Clear the cookies path anytime with the Clear button.

Files & folders

Where Beats stores settings, tools, and your music.

Application data

%APPDATA%\Beats\
├── settings.json
├── liked-songs.json
├── saved-songs.json
├── playlist-orders.json
├── artwork\
└── tools\
    ├── yt-dlp.exe
    ├── ffmpeg.exe
    └── spotdl.exe

Music library (default)

%USERPROFILE%\Music\Beats\
├── My Playlist\
│   └── song.mp3
└── Another Playlist\
    └── ...

The music root folder can be changed in Beats settings. Your playlists remain ordinary folders — back them up, sync them, or move them like any other files.

Troubleshooting

Common issues and how to fix them.

YouTube says "Sign in to confirm you're not a bot"

Set up a YouTube cookies file. Beats also retries with multiple yt-dlp player clients automatically.

Hotkeys don't work

Another app may have registered the same key combo. Use the top chevron tab or on-screen buttons. Only one Beats instance can register hotkeys at a time.

Spotify download is slow the first time

The first Spotify download also installs spotDL and its dependencies. Later downloads are much faster.

Widget is off-screen

Press Ctrl + Shift + \ to reset the widget position and dashboard size to defaults.

Beats won't start — "Already running"

Only one instance can run at a time. Close Beats from Task Manager if the process is stuck in the background.

FAQ

Quick answers to common questions.

Is Beats free?

Yes. Beats is free to download and use on Windows 10 and 11. No account is required.

Does Beats stream music?

No. Beats plays local files from your playlists and downloads new tracks to your disk. Nothing is streamed from the cloud during playback.

Can I use Beats without downloading from the web?

Yes. Add your existing music with Add files or Add folder. The download feature is entirely optional.

Where do downloaded songs go?

Into the playlist folder you chose when pasting the link. Each playlist is a folder under your music root (default: My Music\Beats).

Why does Beats download extra tools on first use?

Web downloads require yt-dlp, ffmpeg, and (for Spotify) spotDL. Beats downloads them automatically so you don't have to install anything manually. They're stored in %APPDATA%\Beats\tools.

Can I move my playlists to another computer?

Yes. Copy your playlist folders (and optionally %APPDATA%\Beats\settings.json) to the new machine. Point Beats at the same music root folder in settings.
